# Planetary-Gear-Calculator
遊星歯車計算機
プラネタリ型、ソーラ型、スター型　それぞれの型から理想の速度伝達比を実現する太陽歯車、遊星歯車、内歯車の歯数を出力します。

入力: 速度伝達比、速度伝達比の許容範囲、遊星歯車の個数

出力："[太陽歯車、遊星歯車、内歯車]:速度伝達比"

計算は以下の歯数条件を下に行います

### 遊星歯車機構における歯数の条件

1. 内歯車 = 太陽 + 遊星*2
    
2. (太陽+内歯車)/遊星の個数=整数

3. 遊星 + 2 < (太陽 + 遊星)*sin(180°/遊星歯車の個数)

注意⚠️: 遊星歯車の不等配位置への配置には対応していません。
